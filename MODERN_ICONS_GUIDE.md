## Best Practices

### 1. Icon Sizing
- **Small icons**: 24-32px (navigation, buttons)
- **Medium icons**: 48-56px (cards, lists)
- **Large icons**: 64-80px (hero sections, main features)

### 2. Consistent Usage
```dart
// Good - Use AppIcons for consistency
Icon(AppIcons.settings)
Icon(AppIcons.home)

// Avoid - Direct Material icons
Icon(Icons.settings)
Icon(Icons.home)
```
