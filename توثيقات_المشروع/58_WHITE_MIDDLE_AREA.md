## 📊 **المواصفات:**

### **المنطقة البيضاء:**
```dart
Positioned(
  top: 400,    // تبدأ من 400px من الأعلى
  bottom: 180, // تنتهي عند 180px من الأسفل
  left: 0,
  right: 0,
  child: Container(
    color: Colors.white, // أبيض نقي 100%
  ),
)
```

**الحساب:**
```
ارتفاع الشاشة: مثلاً 800px
المنطقة البيضاء: 800 - 400 - 180 = 220px
```
